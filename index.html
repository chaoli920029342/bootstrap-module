下拉菜单将下拉菜单触发器和下拉菜单都包裹在 .dropdown 里，加入组成菜单的 HTML 代码。

  • Action
  • Another action
  • Something else here
  • Separated link
  • disabled

1.为 .dropdown-menu 添加 .dropdown-menu-right 类可以让菜单右对齐。

2.为 li标签 添加 .dropdown-header类 可区分 不同的组

3.为 li标签 添加 .divider类 可添加一条分割线

4.为 li标签 添加 .disabled 类 禁用菜单项

5.将 .dropdown 改成 .dropup 类 即为向上弹窗

                                
                                    <div class="dropdown"> 
                                      <button class="btn btn-default dropdown-toggle" type="button" id="dropdownMenu1" data-toggle="dropdown">
                                        Dropdown
                                        <span class="caret"></span>
                                      </button>
                                      <ul class="dropdown-menu" role="menu" aria-labelledby="dropdownMenu1">
                                        <li role="presentation" class="dropdown-header"><a role="menuitem" tabindex="-1" href="#nogo">Action</a></li>
                                        <li role="presentation"><a role="menuitem" tabindex="-1" href="#nogo">Another action</a></li>
                                        <li class="divider"></li>
                                        <li role="presentation"><a role="menuitem" tabindex="-1" href="#nogo">Separated link</a></li>
                                        <li role="presentation" class="disabled"><a role="menuitem" tabindex="-1" href="#nogo">disabled</a></li>
                                      </ul>
                                    </div>

胶囊式标签：增加.nav-pills类，再增加.nav-stacked 类竖直排列

  • Home
  • Profile
  • Messages

1. 添加 .navbar-left类 左浮动，增加.navbar-right类 右浮动

2. 添加 .navbar-fixed-top 类可以让导航条固定在顶部

3. 添加 .navbar-fixed-bottom 类可以让导航条固定在顶部

4. 添加 .navbar-static-top 类即可创建一个与页面等宽度的导航条，它会随着页面向下滚动而消失

5. 添加 .navbar-inverse 类可以改变导航条的外观

                                
                                    <ul class="nav nav-pills nav-stacked" role="tablist">
                                      <li role="presentation" class="active"><a href="#nogo">Home</a></li>
                                      <li role="presentation"><a href="#nogo">Profile</a></li>
                                      <li role="presentation"><a href="#nogo">Messages</a></li>
                                    </ul>
